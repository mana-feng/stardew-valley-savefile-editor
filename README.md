# Stardew Save Editor (Python Version)

星露谷物语存档编辑器 (桌面版)

## 主要功能

- ✅ **角色自定义**：外观、名称、农场名、技能、金钱、职业等。
- ✅ **背包物品管理**：支持 1.6 版本物品，修改数量、品质、添加/移除物品。
- ✅ **配方解锁**：烹饪与打造配方一键解锁。
- ✅ **社交与进度**：修改 NPC 好感度、解锁成就、特殊能力等。
- ✅ **安全备份**：修改前自动备份，防止存档损坏。
- ✅ **本地运行**：无需上传存档，直接读取本地文件。

## 使用说明

1. 确保已安装 Python 3.11+。
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行编辑器：
   ```bash
   python modifier/main.py
   ```
4. 或者直接运行打包好的 `.exe` 文件（如果有）。

## 开发与打包

- **打包 EXE**：运行 `python build_exe.py` 使用 PyInstaller 进行打包。

> [!WARNING]
> 修改存档前请务必备份，不当操作可能导致文件损坏。
